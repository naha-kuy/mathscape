🎯 Panduan Interaktif Petunjuk Konstruksi

  • 📋 Pilih Elemen: Klik checkbox pada panel kanan untuk menampilkan elemen geometri balok 3D
  • 👁️ Mode Tampilan: Gunakan tombol mode tampilan untuk mengubah cara melihat objek
  • 🖱️ Kontrol Kamera: Klik dan geser mouse untuk memutar objek, gunakan scroll untuk zoom
  • 📚 Eksplorasi: Klik header kategori untuk melihat berbagai komponen geometri
  • 🏷️ Label Titik: Titik sudut akan otomatis ditampilkan saat elemen terkait dipilih
